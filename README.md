# Pokémon Explorer Web Application

## 📌 Description

This project is a web application inspired by Pokémon Brilliant Diamond. It allows users to explore Pokémon data such as characters, starter Pokémon, abilities, and stats using a public API. The website will provide an interactive way to search, filter, and view Pokémon information.

## 🎯 Goal

To build a responsive and interactive web application using JavaScript and API integration.

## 🌐 API Used

PokéAPI: https://pokeapi.co/

## 🚀 Features Planned

* Search Pokémon by name
* Filter Pokémon by type (Fire, Water, Grass, etc.)
* Sort Pokémon based on stats (HP, Attack, Speed)
* View detailed Pokémon information (image, abilities, stats)
* Responsive UI design for mobile and desktop

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Fetch API)

## ⚙️ Setup Instructions

1. Clone the repository
2. Open the project folder
3. Run the HTML file in the browser
