# 🌌 Eclipse Legends - Pokémon Explorer Website

## 📌 Description
Eclipse Legends is a web-based Pokémon-inspired application that allows users to explore a custom Pokémon world along with real Pokémon data using PokéAPI.

The website combines:
- A custom game concept (Eclipsia Region)
- Interactive Pokémon search system
- Informational pages about characters, gyms, and mechanics

---

## 🎯 Features

### 🔹 Core Features (Milestone 2 & 3)
- Fetch Pokémon data using PokéAPI
- Display Pokémon dynamically
- Search Pokémon by name or Pokédex number
- Filter by:
  - Region (Kanto → Paldea)
  - Type (Fire, Water, Grass, etc.)
- Sorting options:
  - A-Z
  - Z-A
  - Dex Number (ascending/descending)

### 🔹 Custom Game Features
- Main Characters (Caelum, Lyra, Professor Solara)
- Starter Monsters (Flamix, Aquava, Florion)
- Team Umbra (Villain team)
- Gym Leaders with badges (Ignis, Aquara, Verdan, etc.)
- Battle Mechanics (Mega Evolution, Gigantamax, Z-Moves, Terastallization)
- Region Map (Eclipsia Region)
